ESTRUTURA DE DADOS (animons.json)
================================================================================

Cada animon tem campos:

  id, mr, regional, region, name, type1, type2, species, ability1, ability2, ha,
  height, weight, habitat, feeding, evo_by, inspiration, emoji, image_url,
  description_en, description_pt, deviantart_url, last_updated

Exemplo:
{
  "id": 1,
  "mr": 1,
  "regional": 1,
  "region": "Brasca",
  "name": "Myrmeflag",
  "type1": "Botanical",
  "type2": null,
  "species": "Little Anteater",
  "ability1": "Overgrow",
  "emoji": "🌿",
  "image_url": null,
  "description_en": null,
  ...
}

Obs: image_url e descriptions serão preenchidos via DeviantArt API integration
